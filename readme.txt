Changes to Ygopro
	Updated core engine to 1.033.D
	Online game timer modified. You start with 90 seconds but get a few seconds added for each action you take.

	Custom modifications for generation banlists by Buttys
	Fixed music advantage/disadvantage glitches

	Version 10.8 or higher required
	Tested on macOS Sierra 10.12.2
	Fixed image loading bugs
	Various bug fixes
	-Linux-
	Tested on Fedora 24, Linux Mint 18.1 Cinnamon 64bit
	Instant update problems workaround
